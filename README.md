# land-classification

## Indexes and Classification Methodology
### Water
AWEI_NSH is used to identify water based on findings from [this](https://doi.org/10.1016/j.rsase.2024.101367) source. AWEI_NSH uses the following bands:
- Green (band 3)
- NIR (band 8)
- SWIR1 (band 11)
- SWIR2 (band 12)

### Vegetation Health
NDVI is a common and simple vegetation index used to monitor vegetation health. NDVI uses the following bands:
- Red (band 4)
- NIR (band 8)
